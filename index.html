Simulador de Tienda de Videojuegos

Este simulador permite explorar un catálogo de juegos, buscar productos específicos, agregarlos a un carrito de compras y aplicar códigos de descuento. La interacción principal se realiza a través de la consola JavaScript y cuadros de diálogo como prompt, confirm y alert.

Las funciones principales incluyen: agregar juegos al carrito con agregarAlCarrito(), eliminar elementos del carrito con eliminarDelCarrito(), ver el resumen del carrito con verCarrito(), aplicar descuentos aplicarDescuento(), buscar productos con buscarPorConsola() y finalmente ver todo el cátalgo disponible con mostrarCatalogoConsola().

Los códigos de descuento son:

GAMER10 para un 10% de descuento

WELCOME15 para un 15% de descuento

SUMMER25 para un 25% de descuento
